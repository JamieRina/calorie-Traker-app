# NutriTrack Pro

This project now contains both apps in the same IDE folder:

- `src/` is the Vite React frontend
- `backend/` is the Express + TypeScript + Prisma API

Useful commands from this project root:

- `npm run dev` for the frontend
- `npm run backend:install` to install backend dependencies
- `npm run backend:dev` to run the backend API
- `npm run backend:worker` to run the reminder worker
- `npm run backend:test` to run backend tests
